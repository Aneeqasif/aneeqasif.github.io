# What is DuckDB?

DuckDB is an in-process SQL OLAP database management system. Think of it as "SQLite for analytics"!

## Key Features

- **Fast**: Optimized for analytical queries
- **Portable**: Single file database
- **Standard SQL**: Supports most SQL features
- **Zero dependencies**: Runs everywhere

## Use Cases

Perfect for:
- Data analysis
- ETL pipelines
- Testing and prototyping
- Embedded analytics
